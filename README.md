# Steam Account Cleaner
## С помощью этой проги можно вычиститься так, что организаторы будут думать, что вы легенда, когда вы сисян 100 ранга
### p.s. если вы пользовались ей и вас не нашли, то отпишите в issues опыт и у кого получилось это провернуть))

Утилита для чистки следов Steam аккаунтов на Windows для наших любимых смурфов которые любят играть на тир 4 туриках.

## Что умеет

Находит все аккаунты стима на компе и показывает их в таблице. Можно выбрать нужные и:

- удалить файлы userdata / dota cache (в корзину или совсем)
- вырезать упоминания из логов Steam
- почистить loginusers.vdf и config.vdf
- поменять даты файлов
- посмотреть ранг в доте по account id (через opendota)

Отдельная вкладка со следами — AmCache, Prefetch, MUI Cache, UserAssist, DNS.

## Запуск

Просто скачай exe из [releases](../../releases) и запусти от администратора, иначе часть функций не будет работать (Prefetch).

Windows может ругнуться на SmartScreen — жми "Подробнее" → "Всё равно запустить". Программа подписана, просто сертификат самоподписанный.

## Автообновление

При запуске проверяет версию. Если есть новая — появится плашка в правом верхнем углу, кликни на неё.
